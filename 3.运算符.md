### 运算符
    + 算术运算符
        * 除法 
            + 整除整是整数 
            + 小数除整数是小数
        * 取余数 %
            + 只能针对整数
            + 结果看被除数符号
    + 逻辑运算符
        * 逻辑与 && 
            + 从左到右,只要有一个条件为假，后面的表达式不需要判断(执行),整个条件判断为假
        * 逻辑或 ||
            + 从左到右,只要有一个为真,后面的条件不执行，整个条件判断为真
        * 逻辑非 !
            + 逻辑取反
    + 关系运算符
    + 位运算符
    + 三目运算符
    + 其他运算符
